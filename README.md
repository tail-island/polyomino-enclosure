# polyomino-enclosure

A test environment for NUL programming coucours 2015 problem #2.

## Usage

### Generating Clojure and ClojureScript from *.cljx.

````shell
$ lein cljx once
````

### Generating JavaScript from ClojureScript.

````shell
$ lein cljsbuild once
````

### Open `resources/public/index.html

Push [Execute] button.
